## Dictionaries

### Yomitan

[Yomitan](https://yomitan.wiki/), a browser add-on that allows you to look up Japanese (and Vietnamese) words easily.

- [**※ Yomitan Installation guide for Vietnamese Learning**](https://daihocmo.github.io/learn-vietnamese/yomitan/) - Learn how to setup the best popup dictionary.
- [**※ viet-yomitan**](https://thu-tram.github.io/viet-yomitan/) - A collection of Vietnamese dictionaries for Yomitan.
- [VNEDICT](https://www.mediafire.com/folder/9ss2pn046fcjo/Vi%E1%BB%87t_-_Anh) by Paul Denisowski converted by Marsh Nguyễn for Yomichan.
- [Từ Điển Tiếng Việt Thông Dụng](https://www.mediafire.com/file/ekopqoj0627tpkw/T%E1%BB%AB_%C4%91i%E1%BB%83n_ti%E1%BA%BFng_Vi%E1%BB%87t.zip/file) Vie-Vie dictionary converted by Marsh Nguyễn. The dictionary data is from Từ Điển Tiếng Việt Thông Dụng and was sourced from [https://github.com/vntk/dictionary/tree/master/data](https://github.com/vntk/dictionary/tree/master/data) contains 42012 entries.

## Software

### Desktop

- [※ Anki (Windows, Mac and Linux)](https://apps.ankiweb.net/) - Spaced repetition flashcard program. A must have. It is tricky to use at first.
- [Unikey](https://www.unikey.org/download.html): Vietnamese Keyboard.

### Input Methods

- [UniKey](https://www.unikey.org/) - The most popular, reliable, and standard open-source Vietnamese input method for Windows, developed by Pham Kim Long.
- [EVKey](https://evkeyvn.com/) - A lightweight and highly compatible Vietnamese keyboard software for Windows and macOS designed to fix text-typing bugs in modern applications.
- [Laban Key (Android)](https://play.google.com/store/apps/details?id=com.vng.inputmethod.labankey) - A smart mobile Vietnamese keyboard for Android devices that features advanced text prediction, gesture typing, and high customizability.
- [Laban Key (iOS)](https://apps.apple.com/vn/app/laban-key-g%25C3%25B5-ti%25E1%25BA%25BFng-vi%25E1%25BB%2587t/id916755480) - The official iOS version of the highly popular Laban Key input method, optimized for smooth and efficient typing on Apple devices.
